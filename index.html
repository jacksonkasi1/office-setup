Usage Guide

Getting Started

  1. Include the compiled CSS: dist/output.css
  2. Include the JavaScript utilities: design-system.js
  3. Use design tokens via CSS custom properties or Tailwind classes
  4. Implement components using the provided class names

Build Process

Available npm scripts:

  • npm run build:css - Build CSS once
  • npm run watch:css - Watch and rebuild CSS
  • npm run dev - Start development server
  • npm run build - Build for production
